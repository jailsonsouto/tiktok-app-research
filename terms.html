1. Project Description

The "Research-Tool-DSA" is a non-commercial academic application developed as part of a Postgraduate MBA in Data Science and Analytics. The tool is designed to facilitate Netnography and Sentiment Analysis (ABSA/ASTE) within the cosmetics industry for academic and statistical purposes.

2. Acceptance of Terms

This application is a private research tool. Access is restricted to the lead researcher and academic supervisors associated with the MBA program. It is not intended for public or commercial distribution.

3. Data Processing & TikTok API

We utilize the TikTok for Developers API to access public content and strictly comply with TikTok’s Developer Terms and applicable policies as described on TikTok for Developers . This tool does not "scrape" data; it requests it through official channels for scientific analysis of consumer behavior and Natural Language Processing (NLP) modeling.

4. Ethical Guidelines & Data Usage

  • Data is used exclusively for the MBA thesis/capstone project.
  • No personal data is used to identify individuals.
  • Results are presented only in aggregate form (charts, trends, sentiment scores).
  • Commercial exploitation of the collected data is strictly prohibited.
